- multi-agent : 멀티에이전트 협업/오케스트레이션/상태머신
- onprem      : 폐쇄망/로컬추론/vLLM 서빙/GPU 최적화
- finetuning  : QLoRA/도메인 미세조정/경량화
- rag         : 검색증강생성/벡터DB/지식 파이프라인
- agentops    : 상태관리/비용통제/장애복구/관측성

[출력 형식 — 매우 중요]
- 오직 유효한 JSON 객체 1개만 출력한다. 마크다운 코드펜스(```), 설명 문장, 앞뒤 텍스트 절대 금지.
